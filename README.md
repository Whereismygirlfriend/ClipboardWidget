# 클립보드 위젯

클립보드에 복사한 텍스트, 이미지, 파일 경로를 자동으로 기록하고 관리하는 데스크톱 프로그램입니다.  
작업 중 복사 이력을 빠르게 확인하고, 필요한 데이터만 골라 내보낼 수 있습니다.

## 주요 기능

- 텍스트 / 이미지 / 파일 경로 자동 감지 및 기록
- 이미지 원본 바이트 우선 저장으로 해상도 유지 캡처
- 중복 감지(해시 기반) 및 중복 여부 표시
- 실시간 목록 반영 + 자동 동기화 보정
- 타입 필터(전체/텍스트/이미지/파일), 중복 필터, 검색
- 체크 선택 후 일괄 삭제
- 더블클릭 상세 보기
- 내보내기 지원
- `TXT`, `Markdown`, `PDF`, `PNG 폴더`, `PNG ZIP`
- 백그라운드 모드(트레이 상주) 지원
- 다크/라이트/커스텀 테마 지원
- 단축키 커스터마이징 지원
- 실시간 저장 모드(지정 폴더로 자동 저장)

## 사용 방법

1. 프로그램 실행
2. 평소처럼 텍스트/이미지/파일을 복사
3. 목록에서 항목 확인
4. 필요 시 필터/검색으로 항목 정리
5. 체크 후 삭제하거나 원하는 형식으로 내보내기

## 데이터 저장 위치

기본 저장 경로:
`C:\Users\<사용자명>\AppData\Local\ClipboardManager`

주요 파일/폴더:
- `clipboard_history.db`: 클립보드 기록 DB
- `user_settings.json`: 사용자 설정
- `images\...`: 캡처된 이미지 원본 파일
- `imports\...`: JSON 가져오기 시 복사된 이미지

## 참고 사항

- 같은 내용이 매우 짧은 시간 안에 반복 복사되면 중복 방지 로직으로 기록이 생략될 수 있습니다.
- 원본 앱이 클립보드에 축소본 이미지를 넣는 경우, 프로그램도 해당 데이터 기준으로 저장합니다.
- 백그라운드 모드가 켜져 있으면 창을 닫아도 트레이에서 계속 동작합니다.
